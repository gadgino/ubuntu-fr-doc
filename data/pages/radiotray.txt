====== RadioTray : écoutez vos radios simplement ! ======

Radio Tray est un logiciel extrêmement léger qui permet d'écouter des radios en ligne. Il s'intègre dans le tableau de bord et permet notamment la gestion de vos radios préférées grâce à des actions basiques telles que :
  * Ajouter/ Supprimer des radios.
  * Renommer des radios.
  * Créer des groupes.

Il ne se présente que sous la forme d'une icône dans le tableau de bord mais peut s'ouvrir avec une fenêtre pour les réglages.

{{ :radiotray_natty_01.png }}

===== Pré-requis =====
 
  * Disposer des [[:sudo|droits d'administration]].
  * Disposer d'une connexion à Internet configurée et activée.

===== Installation =====

**Radio Tray** est **disponible dans les dépôts**, il vous suffit donc d'[[:tutoriel:comment_installer_un_paquet|installer le paquet]] :  **[[apt>radiotray|radiotray]]**.

Si vous souhaitez que Radiotray démarre automatiquement : ouvrez [[:tutoriel:application_demarrage|Application au démarrage]] cliquer sur ajouter, dans la case "Nom" taper "radiotray", dans la case "Commande" taper "/usr/bin/radiotray", dans "Commentaire" entrez ce que vous voulez (ex: "radio" ou "web radio").

=====Utilisation=====
==== Particularité sous Gnome-shell ====
Sous Ubuntu-Gnome, la **zone de notification** est **masquée** ! 

Pour la **voir** il faut **amener la souris en bas de l'écran** ce qui permet ensuite de voir l'icône de **radiotray**.

==== Ajout de radios ====
Vous pouvez ajouter des radios dans "Préférences/Configurer les radios..." mais à l'aide de cette fonction vous ne pouvez ajouter qu'une seule radio à la fois.

Cependant en éditant le fichier **bookmarks.xml** situé dans **~/.local/share/radiotray** vous pourrez ajouter en masse les radios et personnaliser votre menu. Une **liste de radios françaises** (màj 07/04/2013) existe que vous pouvez ajouter en suivant les instructions ci-dessous :
 
  - Sauvegardez votre ancien bookmarks.xml avec, par exemple, avec la commande : <code>cp ~/.local/share/radiotray/bookmarks.xml ~/.local/share/radiotray/bookmarks00.xml</code>
  - Cliquez sur "bookmarks.xml" au dessus de la liste des radios ;
    * le **fichier** est **automatiquement ouvert**
    * il ne vous reste **plus qu'à l'enregistrer et écraser l'ancien** dans le dossier caché de votre dossier personnel :\\ **//.local/share/radiotray/bookmarks.xml//**.
  - Allez ensuite dans //Préférences// -> //Recharger les signets//.

<note tip>Si vous souhaitez que cette liste de radios soit proposée par défaut pour les autres sessions, il vous faudra la copier (avec les droits administrateurs) dans le dossier /usr/share/radiotray : <code>sudo cp ~/.local/share/radiotray/bookmarks.xml /usr/share/radiotray/bookmarks.xml</code></note>

Si vous avez besoin d'aide pour modifier le fichier **bookmarks.xml**, allez consulter le sujet [[http://forum.ubuntu-fr.org/viewtopic.php?id=547881&p=2|bookmarks.xml pour radiotray]]. Une liste de radio prête à être intégrée dans Radio Tray est disponible en fin de page.


===== Problèmes connus =====

==== Icône de notification s'affiche mal ("Icône vide") ====

Pour [[Trusty|Ubuntu 14.04]] sous les environnements Unity, KDE et XFCE  une solution est de [[:tutoriel:comment_modifier_un_fichier|modifier le fichier]] **config.xml** du répertoire caché **~/.local/share/radiotray** de l'application Radiotray. 


{{ :media:radiotrayconfig.png?nolink |}}

  * Dans le fichier ligne 9, remplacer dans l'intitulé : <file>value="appindicator"</file> par "systray" ou "systray" par "appindicator" selon le cas de figure rencontré au départ.
  * Enregistrer le fichier de configuration puis le fermer.
  * Redémarrer Radiotray.

==== Son de mauvaise qualité ====

Cela peut être du à un réglage trop élevé du volume sonore défini par RadioTray, ce qui peut provoquer des saturations. Vous pouvez modifier le volume directement dans les réglages du son lorsque RadioTray fonctionne, en allant dans les préférences du son, vous avez un onglet Applications, dans lequel vous pourrez régler le niveau sonore pour chaque application (à ne pas confondre avec la possibilité de régler le son dans votre application elle même).

Il est possible que le flux radio soit de faible qualité (suffisant pour les conversations, insuffisant pour la musique). Allez voir sur le site web de la radio concernée si d'autres adresses de flux sont disponibles, si c'est le cas vous pourrez ensuite la modifier dans le fichier "bookmarks.xml" (voir ci-dessus).
==== Absence de fichier bookmarks.xml ====

Avec Ubuntu-12.04 (et 14.04), pour que le fichier bookmarks.xml soit présent dans ".local/share/radiotray", il faut avoir lancé RadioTray une fois afin qu'il crée ce fichier automatiquement (il y écrit une liste prédéfinie de radios anglophones).

==== Radiotray ne démarre pas ====

Si avec Ubuntu-16.04, lancer radiotray dans terminal affiche l'erreur « OSError: [Errno 2] Aucun fichier ou dossier de ce type: '/usr/bin/src' ».
Il suffit d'installer le paquet [[apt>python-xdg|python-xdg]] [[https://bugs.launchpad.net/ubuntu/+source/radiotray/+bug/1557115/comments/10|source #LP 1557115]]

==== Radiotray affiche une icône noire inutilisable ====

Si avec Ubuntu-16.04, lancer radiotray affiche une icône noire inutilisable dans la barre d'icônes, installer le paquet [[apt>gir1.2-appindicator3-0.1]] [[https://bugs.launchpad.net/ubuntu/+source/radiotray/+bug/1557115/comments/10|source #LP 1557115]]

==== Interruptions de 5 secondes ====
Si Radiotray interrompt la réception des radios pendant 5 secondes, alors que ni votre connexion internet , ni la web-radio elle-même (vérifier sur le site de celle-ci si le son s'interrompt parfois) ne sont en cause, voici la solution.

Éditez avec les droits root le fichier ''/usr/lib/python2.7/dist-packages/radiotray/AudioPlayerGStreamer.py'' et commentez (à l'aide d'un #) la ligne 156.
  sudo geany /usr/lib/python2.7/dist-packages/radiotray/AudioPlayerGStreamer.py

La ligne 141 ou 156 (suivant la version) :
<code>
                self.player.set_state(Gst.State.PAUSED)
</code>
doit devenir :
<code>
                #self.player.set_state(Gst.State.PAUSED)
</code>
