======Trucs et astuces pour utiliser Libreoffice======

Cette page rassemble quelques astuces, procédures, souvent basiques, mais qui pourraient vous faire perdre du temps à les chercher par vous-même.
N'hésitez pas ajouter les vôtres, si vous estimez qu'elles peuvent être fréquemment utilisées.

===== Compatibilité avec Microsoft Office =====
Si vous souhaitez travailler sur des documents provenant de (ou à destination de) la suite bureautique Microsoft Office, //consultez le tutoriel **[[:tutoriel:compatibilite_libreoffice_et_msoffice|Comment améliorer la compatibilité entre Libreoffice et MS Office]]**.//

===== Numéroter ses pages =====
==== Méthode simple et officielle ====

    * Allez dans //Insertion → Pied de page → Tout ou Standard//.
    * Cliquez sur le petit encadré //Pied de page//.
    * Allez dans //Insertion → Champ → Numéro de page//.
    * Vous pouvez placer ces numérotations ou vous le voudrez, elles se feront automatiquement sur chaque pages.

==== Numéroter à partir de la 2e page ====
Les solutions pour avoir une première page sans numérotation :

  * **Vous pouvez voir pour les [[http://user.services.openoffice.org/fr/forum/ftopic11549.html|Facilitateurs de numérotation de page]]**, ce sont des extensions à installer dans Libreoffice. (Voir comment [[:libreoffice#installer_une_extension|installer une extension]]).

Solution manuelle :
  * Sur la première page : 
        * Aller dans //Format → Style et formatage → onglet Style de page → Première page//
  * Sur la deuxième page :
        * Allez dans //Insertion → Pied de page → Standard//
        * Cliquez sur le petit encadré pied de page
        * Allez dans //Insertion → Champ → Numéro de page//
        * Double-cliquer sur le numéro de page que vous venez d'insérer, puis mettre -1 dans //Correction//.

===== Utiliser sa tablette graphique =====

Si vous disposez d'une [[:tablette graphique]], vous pouvez l'utiliser pour écrire dans un document, par exemple pour y apposer votre signature. Il suffit pour cela de cliquer sur l'icône « **Afficher les fonctions de dessin** » dans la barre supérieure, ou de passer par le menu **Affichage > Barres d'outils > Dessin**, puis de sélectionner dans la barre qui apparaît l'outil « **ligne à main levée** ». Chaque trait que vous ferez avec le stylet sera une nouvelle [[http://fr.wikipedia.org/wiki/Image_vectorielle|image vectorielle]].

=====Se connecter à des bases de données =====

Pour effectuer un publipostage ou mailing, le paquet **[[apt>libreoffice-base]]** doit être installé, faute de quoi les boutons "définir" restent grisés quand on tente de se connecter à un fichier de base de donnée

Il existe également des extensions pour augmenter la connectivité de LibreOffice.

Pour ce connecter à une base mysql ou autre [[:libreoffice_mysql|voir la page décrivant la méthode.]]
